## Setup
1. Clone the repo: `git clone https://github.com/NickBell123/Poker-Bot.git`
2. Set up a virtual environment: `python -m venv venv`
3. Activate: `.\venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install pandas`
